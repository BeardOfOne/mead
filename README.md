<h1>Gosling Engine</h1>
<img src="https://travis-ci.org/danielricci/gosling-engine.svg?branch=master" />
This is the repository for the Einstein Assets Editor, a tool used to create tilemaps and export the data into a game.

<h3>Version</h3>
<p>There is a <b>three-week</b> window for new release and/or patch fixes when new feature/enhancement requests or bug fixes get approved. When there are no tasks to work on, there will not be any new versions released.</p>

<p>Please see the releases section and read the release notes in the wiki for more details</p>
